Underlying data
The Freizeitkarte maps (edition 14.08) are based on OpenStreetMap data from 08/09/2014. Thanks to all the mappers for their contribution to the OSM project.

Change log for release 14.08:
- general improvement of the map data (by the OSM project)
- general improvement of the map design
+ modified design of garages areas (landuse=garages)
+ separated design of ruins (building=yes + ruins=yes)
+ additional framing for residential and industrial areas
+ design of hedge banks (barrier=hedge_bank)
+ changed design of chair lifts (aerialway=chair_lift)
+ design of private swimming pools (leisure=swimming_pool + access=private)
- additional theme (map design) "fzk-outdoor-contrast" (thanks to Stephan)

Next release:
The next release is planned for October 2014.

General
The Freizeitkarte maps always consist of two parts:
- the map data (*.map)
- the map design (theme)

Download
The download of a Freizeitkarte map or a theme is possible in two ways:

  direct to your personal computer →         
  direct into the Android app Locus Map →

Hints
- When downloading into Locus Map a large amount of data will be loaded into your Android device.
- You start the download to Locus Map by choosing the appropriate link in your Android browser.

Installation
Depending on the type of download and the used app the following applies:

    map and design are to be transferred to the Android device; finally you have to choose the design to be used
nothing to do, you can already use the map

Install Freizeitkarte map under Locus Map (manually)
- unzip map download
- copy Freizeitkarte map (*.map) to directory "Locus/mapsVector"

Install design under Locus Map (manually)
- unzip theme download
- copy theme directory "freizeitkarte" to directory "Locus/mapsVector/_themes"

Use a Freizeitkarte map with Locus Map
- choose map: Map icon → vector maps → choose directory "Locus/mapsVector" → choose Freizeitkarte map
- choose theme: choose map as before → details → external themes → choose "freizeitkarte"

Install map under OruxMaps
- unzip map download
- copy Freizeitkarte map (*.map) to directory "oruxmaps/mapfiles"

Install theme under OruxMaps
- create a subdirectory "/mapstyles/FZK" within directory "oruxmaps"
- copy the content of the theme into the new created subdirectory
The result should look like this:
- oruxmaps/mapstyles/FZK/symbols/*.png
- oruxmaps/mapstyles/FZK/patterns/*.png
- oruxmaps/mapstyles/FZK/freizeitkarte.xml

Use a Freizeitkarte map with OruxMaps
- Map icon → offline maps, choose → Freizeitkarte
- Settings → Modify → Mapsforge theme → choose freizeitkarte.xml
