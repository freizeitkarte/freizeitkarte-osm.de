General
The Freizeitkarte maps always consist of two parts:
- the map data (*.map)
- the map design (theme)

Download
The download of a Freizeitkarte map or a theme is possible:

direct to your personal computer
direct into the Android app Locus Map
direct into the Android app OruxMaps

Hints
- When downloading into Locus Map or OruxMaps a large amount of data will be loaded into your Android device.
- You start the download to Locus Map or OruxMaps by choosing the appropriate link in your Android browser.

Installation
Depending on the type of download and the used app the following applies:

map and design are to be transferred to the Android device; finally you have to choose the design to be used
map + theme are loaded together; after that there is nothing to do, you can already use the map
you have to load map + theme in two steps; finally you have to choose the design to be used

Install map under Locus Map (manually)
- unzip map download
- copy map (*.map) to directory "Locus/mapsVector"

Install theme under Locus Map (manually)
- unzip theme download
- copy theme directory to directory "Locus/mapsVector/_themes"

Use a map with Locus Map
- Map icon → OFFLINE → directory MAPSVECTOR → choose map

Use a theme with Locus Map
- Map config icon → MAP THEMES → EXTERNAL THEMES → choose theme

Install map under OruxMaps (manually)
- unzip map download
- copy map to "oruxmaps/mapfiles"

Install theme under OruxMaps (manually)
- unzip theme download
- copy directory to "oruxmaps/mapstyles"

Use a map with OruxMaps
- Map icon → Switch map → OFFLINE → choose map

Use a theme with OruxMaps
- Map icon → Mapsforge theme → choose theme
