Underlying data:
The Freizeitkarte maps are based on OpenStreetMap data of the July 16st, 2017.
Thanks to all the mappers who contributed data.

Additions / modifications in Release 17.07:

  • general improvement of the map data (by the OSM project)

Information:
For users on Microsoft Windows with BaseCamp we recommend to use the new GMAP Installers. The reason for this is that the oldfashioned installer do not display height profiles in BaseCamp. You can read more about how to use these GMAP Installers on the Installation page.

Next release:
The next release is planned for begin of October 2017.
