Underlying data:
The Freizeitkarte maps are based on OpenStreetMap data of the November 8th, 2015.
Thanks to all the mappers who contributed data.

Additions / modifications in Release 15.11:

  • general improvement of the map data (by the OSM project)
  • version update of the tools used in mt.pl
  • support for OS X and additional linux shells in ReplaceTyp.sh

Important Information:
In this release we can't provide the automatic installer for DEU+ on BaseCamp under Microsoft Windows.
The size has exceeded the maximum possible for on automatic installer. As an alternative we recommend to
download the install archive for BaseCamp under Apple Mac OS X. Extracted and moved to the correct location
it also works properly with BaseCamp under Microsoft Windows.
We are working on it.

Next release:
The next release is planned early 2016.
