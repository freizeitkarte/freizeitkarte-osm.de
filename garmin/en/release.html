Underlying data:
The Freizeitkarte maps are based on OpenStreetMap data of the January 11th, 2015.
Thanks to all the mappers who contributed data.

Additions / modifications in Release 15.01:

  • General improvement of the map data (by the OSM project)
  • Routing improvement regarding OSM tags trunk/motorway/motorroad
  • Additional language French and changing default language for the respective maps (FRA, FRA+, LUX and BEL)
  • Additional maps for US Regions in the maps directory
  • Unifying labels of different paths and tracks
  • Development Environment now supports different selectable mkgmap styles and the building of Unicode maps

Advice for Monterra owners:
There are known issues regarding the Monterra. Looks like not objects are visible in all zoom factors. This issue might also affects other Series (Nuvi, Zumo). Feedback about this is very welcome.

Next release:
The next release is planned for January 2015.
